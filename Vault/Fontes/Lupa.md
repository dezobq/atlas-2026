# Lupa (Agência Lupa)

Primeira agência brasileira de fact-checking, fundada em 2015. Signatária do código de princípios da International Fact-Checking Network (IFCN). Parceira atual do UOL.

## Como referenciar no Atlas

- **Classificação literal** preservada (ex: "Falso", "Verdadeiro", "Exagerado", "Contraditório")
- **URL primária** sempre arquivada via Wayback Machine antes de publicar
- **Atribuição:** "Veredito: Agência Lupa, <data>, <url-arquivada>"

## Formato típico de URL de veredito

```
https://lupa.uol.com.br/jornalismo/YYYY/MM/DD/<slug-do-checagem>
```

## Categorias usadas (vocabulário Lupa, não normalizar)

- Verdadeiro
- Verdadeiro, mas
- Ainda é cedo
- Exagerado
- Contraditório
- Insustentável
- Falso
- De olho

## Cuidados editoriais

- Atribuir SEMPRE — não citar veredito sem link arquivado
- Não traduzir classificação ("Falso" ≠ "False" — manter pt-BR literal)
- Se Lupa retratar, atualizar no Atlas com errata pública

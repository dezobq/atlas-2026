# Projeto Comprova

Consórcio brasileiro de jornalismo colaborativo para verificação. Reúne dezenas de veículos jornalísticos. Foco especial em conteúdo viral em redes e desinformação em períodos eleitorais.

## Como referenciar no Atlas

- **Classificação literal** preservada
- **URL primária** arquivada via Wayback
- **Atribuição:** "Veredito: Projeto Comprova, <data>, <url-arquivada>"

## Formato típico de URL

```
https://projetocomprova.com.br/publicações/<slug>/
```

## Categorias usadas

- Comprovado
- Falso
- Enganoso
- Sátira
- Distorcido
- Contraditório
- Verificações sobre vídeo/imagem manipulados

## Cuidados editoriais

- Comprova é especialmente útil para conteúdo de redes sociais (posts virais, vídeos curtos)
- Mesma declaração pode ter veredito de Comprova **e** Lupa **e** Aos Fatos — listar todos (ver [[../Dominios/Cascata-de-Vereditos]])
- Por ser consórcio, atribuir como "Projeto Comprova" (não veículo individual), salvo se o veredito for marcadamente de um único parceiro

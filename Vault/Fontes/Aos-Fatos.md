# Aos Fatos

Agência de fact-checking brasileira, fundada em 2015. Signatária IFCN. Desenvolve também o bot "Fátima" para checagem assistida.

## Como referenciar no Atlas

- **Classificação literal** preservada
- **URL primária** arquivada via Wayback Machine
- **Atribuição:** "Veredito: Aos Fatos, <data>, <url-arquivada>"

## Formato típico de URL

```
https://www.aosfatos.org/noticias/<slug-da-checagem>/
```

## Categorias usadas

- Verdadeiro
- Falso
- Exagerado
- Distorcido
- Contraditório
- Insustentável
- Subestimado
- Sátira (usado quando contexto humorístico é central)

## Cuidados editoriais

- Aos Fatos publica também análises mais longas — só agregar a parte "veredito" classificada
- Se houver retração ou nota da equipe, atualizar errata no Atlas
